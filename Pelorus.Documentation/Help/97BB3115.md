# RssFeed Class
 

Data contract for an RSS feed. This model is compliant with the RSS 2.0 specification as defined by the Berkman Center at Harvard University. The documentation of this data contract is based primarily on the online documentation provided by the Berkman Center. http://cyber.law.harvard.edu/rss/
